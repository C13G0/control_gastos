# Proyecto: Control de Gastos Personales

Aplicación de consola desarrollada en Python que permite registrar y consultar gastos personales organizados por mes y categoría.

## Características

- Registrar gastos con fecha, descripción, categoría y monto
- Consultar gastos por mes
- Ver reporte mensual con total gastado

## Requisitos

- Python 3.x
- pip

## Instalación

1. Clona el repositorio:
git clone https://github.com/C13G0/control_gastos.git

2. Entra a la carpeta del proyecto:
cd control_gastos

3. Crea el entorno virtual:
python -m venv venv

4. Activa el entorno virtual:
venv\Scripts\activate

5. Instala las dependencias:
pip install -r requirements.txt

## Ejecución

cd src
python main.py
